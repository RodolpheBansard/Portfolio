Lumiplay

Lumiplay est une solution SaaS permettant de crée, gérer et planifier des contenus. Ces contenus sont ensuite envoyé sur leur support d’affichage (application mobile, panneau d’information, transport en commun, ...)

J’ai pu participer au développement de cette application pendant un an en utilisant Angular pour le frontend et Java (Springboot) pour le backend avec une base de données PostrgesSQL
